Website Style Guide

by Stev

I'm still very new to creating websites, but this is to collect my thoughts on what "my style" is.

Placement

I like a central content divider placed over a larger surrounding image. This can be done by:

  • set a background image in the body, fix its position and set it to cover
  • set the body to flex around the center
  • set the width of your content div to be roughly 50-60%

This should look like this:

body { background-image: url("background.jpg"); background-size: cover; background-position: center; background-repeat: no-repeat; background-attachment: fixed; } div { display: flex; justify-content: center; align-items: center; }
