Colour

Colour is an important part of website design. It can be tough to pick the right colour theme but applying the theme to your site is straightforward.

How to describe colours in code

There are many ways we can use our code to add colour but first we need to be able to identify the colours we want in our code. There are four main ways we can identify colours in our code. :

  • colour names
  • hexcodes
  • rgb
  • hsl

Colour names

There are 147 predefined color names. These allow you to write out a colour name to identify a colour in your code. You might want your text to be lightsalmon on a navajowhite background. You could choose tomato to emphasise certain sections. All you have to do is refer to one of the predefined colour names in your code.

Hexcodes

Hexcodes are a way of converting colours into a 6 digit hexidecimal number prefaced with a hash. These can be used to describe the predefined colours like skyblue (#4682b4) or orchid (#da70d6). The real advantage over colour names though is that we can use colours outside the list of 147 predefined colours. We can use #937ac3, #0c8927 or #6786d6. If your hexidecimal value is made up of 3 matching pairs we can shorten these to 3 single characters. So #aa5522 and #a52 are interpretted the same.

RGB

RGB allows us to define colours by giving values for Red, Green and Blue which when combined create colour. This is done by using the rgb(r,g,b) function: where r, g and b are numbers between 0 and 255 inclusive. An example of this would be rgb(120,165,23) or rgb(197,76,24.

HSL

HSL works on a similar method to RGB, but instead we define Hue, Saturation and Lightness. This uses the hsl(h,s,l) function: where h is between 0 and 360, and s & l are percentages. An example of this would be hsl(124,74%,32%) or hsl(334,81%,29%).
